# Canopy Connect (canopy)
Canopy Connect is an insurance infrastructure platform that lets consumers and businesses quickly and securely share property and casualty insurance information through integrations with 400+ carriers covering 95%+ of the U.S. auto and homeowners markets. The API returns structured policy, driver, vehicle, claims, and property data in seconds, replacing manual verification workflows used across mortgage lending, auto finance, insurance carriers, and embedded insurance products.

## APIs

### Canopy Connect API
The Canopy Connect API returns structured property and casualty insurance data directly from 400+ carriers in real time. Applications can verify coverage, retrieve policy documents, pull driver and vehicle details, and surface claims history in under a minute instead of the typical 30-minute manual verification process.

**Human URL:** [https://www.usecanopy.com/api](https://www.usecanopy.com/api)

#### Features

- 400+ insurance carrier integrations (95%+ market coverage)
- 96% auto insurance coverage, 91% homeowners coverage
- 500+ structured insurance data fields returned in seconds
- Policy details, coverage limits, and deductibles
- Driver information and licensed operators
- Vehicle data including VIN and coverage
- Claims history
- Property assessment data
- Policy document retrieval
- Drop-in SDK for rapid embedded integration
- SOC 2 Type 2 certified infrastructure
- 256-bit AES at rest, TLS 1.3+ in transit

#### Use Cases

- Mortgage lending insurance verification

- Auto finance proof-of-insurance workflows
- Driver compliance monitoring
- Insurance carrier quoting and underwriting
- Embedded insurance in consumer applications
